Why browser-sync?

Everything you need, nothing you don't.

🔀

Any direction

Chrome → Firefox, Firefox → Brave, Edge → Vivaldi — every combination works both ways.

☁️

Sync-safe

Preserves root GUIDs and writes tombstones so browser sync services don't push spurious deletions.

📦

Zero dependencies

Pure Python 3, which ships with macOS. No pip, no venv, no Node — just one script.

💾

Auto-backup

Creates a timestamped backup of the destination before writing. Restores it automatically if anything fails.

👁️

Dry-run mode

Preview exactly what would change — no file is touched until you explicitly confirm.

🌐

English & Spanish

UI language is detected automatically from your system locale. More languages are easy to add.

How it works

Three steps, one command.

1

Select source & destination

An interactive menu lists every installed browser and each of its profiles. Pick one as source, one as destination.

2

Review the summary

browser-sync shows you exactly what will be written and asks for confirmation. Use --dry-run to skip even that.

3

Bookmarks are transferred

All data is normalised through an internal Chromium JSON pivot. A backup of the destination is created first, and restored automatically if anything goes wrong.

Usage

Interactive by default, scriptable when needed.

$ browser-sync # interactive mode — pick source & destination
$ browser-sync --dry-run # preview changes, write nothing
$ browser-sync --help # show all options

Installation

One tap, one install.

1 — Add the tap
$ brew tap lcajigasm/browser-sync
2 — Install
$ brew install browser-sync
3 — Run
$ browser-sync

Requires macOS Monterey or later and Python 3.10+, which ships with macOS — no extra installs needed.
